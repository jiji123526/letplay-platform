✅ 채널이 생성되었습니다!
채널 관리자 기능을 확인하세요.
  • 우측 상단 ⋮ 메뉴를 누르면 "관리자 설정"이 나타납니다. 여기서 모든 채널 설정을 관리할 수 있습니다.
  • 채널 이름과 프로필 사진을 변경합니다. 사진은 정사각형으로 잘라서 업로드됩니다. 변경 즉시 모든 접속자에게 반영됩니다.
  • 방문자에게 보여지는 이용 규칙을 설정합니다. 여러 섹션으로 나눌 수 있으며, 규칙이 있으면 채팅 상단에 ℹ️ 버튼이 나타납니다.
  • 채팅 상단에 배너 형태로 표시됩니다. 제목과 내용을 작성할 수 있으며, 사용자가 닫을 수 있습니다. 새 공지를 등록하면 닫았던 배너가 다시 나타납니다.
  • 채팅 말풍선의 기본 색상을 변경합니다. 7가지 프리셋 중 선택하거나 직접 색상을 지정할 수 있습니다. 사용자도 개인 설정에서 변경 가능합니다.
  • 비밀번호를 설정하면 채널 입장 시 비밀번호를 입력해야 합니다. 비밀번호를 아는 사람만 채팅에 참여할 수 있습니다. 비워두면 누구나 입장 가능합니다.
  • 특정 단어가 포함된 메시지를 자동으로 차단합니다. 영구 또는 기간 한정(1일/7일/30일)으로 설정 가능합니다. 서버에서 검사하므로 우회할 수 없습니다.
    • 신고 방법: 일반 사용자가 메시지를 꾹 누르면 "신고" 버튼이 나타납니다
    • 신고 확인: 신고된 메시지는 관리자 채팅에 🚨 표시와 함께 나타납니다. 해당 메시지를 탭하면 원본으로 이동합니다
    • 채팅에서 차단: 관리자가 메시지를 꾹 누르면 "사용자 차단" 버튼으로 즉시 차단할 수 있습니다
    • 차단 효과: 차단된 사용자는 메시지를 보낼 수 없으며, UID + 기기 지문으로 식별됩니다
    • 이의 제기: 차단된 사용자는 1회 DM(비밀 메시지)을 보낼 수 있습니다. 관리자만 볼 수 있습니다
    • 이의 제기 토글: 관리 → "이의 제기 허용 중"을 눌러 기능을 끌 수 있습니다. 끄면 차단된 사용자는 아무것도 보낼 수 없습니다
    • 비밀 메시지 토글: 관리 → "비밀 메시지 허용 중"을 눌러 기능을 끌 수 있습니다. 끄면 일반 사용자의 + 메뉴에서 DM 버튼이 사라집니다
    • 차단 해제: 관리자 설정 → 관리 → 차단 사용자에서 차단 목록을 확인하고 해제할 수 있습니다
    • 메시지 삭제: 신고된 메시지는 관리자가 꾹 눌러 삭제할 수 있으며, 답장도 함께 삭제됩니다
  • 채팅을 일시적으로 중단합니다. 관리자만 메시지를 보낼 수 있고, 일반 사용자는 DM만 가능합니다. 모든 사용자에게 알림이 표시됩니다.
  • 임시 채팅 세션을 시작합니다. 참여자들에게 팝업이 표시되며, 종료 시 모든 라이브 메시지가 자동 삭제됩니다. 이모지 리액션 바가 활성화됩니다.
💡 채널 주소를 공유하면 누구나 익명으로 참여할 수 있습니다. 로그인 없이 바로 채팅이 가능합니다.
